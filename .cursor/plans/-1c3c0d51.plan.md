# План: Приведение имен файлов изображений к единому формату

## Проблемы с именами файлов

### 1. winter/bright - отсутствуют стандартные файлы

- Есть только: `bright winter.webp` (с пробелом)
- Нужно: `man-young.webp`, `man-old.webp`, `woman-young.webp`, `woman-old.webp`

### 2. Опечатки в именах файлов

**spring/light:**

- `man-yong.webp` → `man-young.webp`

**summer/light:**

- `woman_young.webp` → `woman-young.webp`

**summer/cool:**

- `woman-yong.webp` → `woman-young.webp`

**summer/soft:**

- `man.webp` → определить и переименовать в `man-young.webp` или `man-old.webp`

**autumn/soft:**

- `woman-yong.jpg` → `woman-young.webp` (также конвертировать в webp)
- `young-man.webp` → `man-young.webp`

**autumn/warm:**

- `man-yong.webp` → `man-young.webp`
- `waoman-old.webp` → `woman-old.webp`

## Стандартный формат имен файлов

Все файлы должны иметь формат:

- `man-young.webp` - мужчина, молодой
- `man-old.webp` - мужчина, пожилой
- `woman-young.webp` - женщина, молодая
- `woman-old.webp` - женщина, пожилая

Предпочтительный формат: `.webp` (если файл в `.jpg`, конвертировать в `.webp`)

3. Конвертировать .jpg в .webp где необходимо
