# Инструкция по перемещению папки фото

Из-за проблем с кодировкой в PowerShell, выполните одно из следующих действий:

## Вариант 1: Через командную строку (cmd)

1. Откройте командную строку (cmd) в папке проекта
2. Выполните команды:

```cmd
mkdir public
xcopy /E /I /Y фото public\фото
rmdir /S /Q фото
```

## Вариант 2: Через Node.js скрипт

1. Откройте терминал в папке проекта
2. Выполните:

```bash
npm run move-photos
```

Или напрямую:

```bash
node scripts/move-photos.js
```

## Вариант 3: Вручную через Проводник Windows

1. Создайте папку `public` в корне проекта
2. Перетащите папку `фото` в папку `public`
3. Удалите исходную папку `фото` из корня проекта

## Проверка

После перемещения структура должна быть:

```
funel_ganza/
├── public/
│   └── фото/
│       ├── hoodie_*.png
│       └── цветотипы/
│           ├── autumn/
│           ├── spring/
│           ├── summer/
│           └── winter/
```

Исходная папка `фото` в корне проекта должна быть удалена.
